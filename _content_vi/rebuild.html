Kể từ Go 1.21, bộ công cụ nhị phân của Go được phục vụ tại go.dev/dl/ có thể được tái tạo và xác minh bởi bất kỳ ai, trên bất kỳ nền tảng nào, sử dụng golang.org/x/build/cmd/gorebuild. Để biết thêm chi tiết, xem bài blog "Toolchain Go có thể tái tạo và xác minh hoàn hảo."

Trang này được cập nhật hàng ngày với kết quả chạy gorebuild trong máy ảo Ubuntu, sử dụng script sau: 

apt-get update &&
apt-get -y install software-properties-common &&
add-apt-repository universe &&
apt-get -y install golang-go msitools &&
go run golang.org/x/build/cmd/gorebuild@latest -p=4

Cài đặt msitools cho phép gorebuild kiểm tra nội dung của tệp cài đặt Windows MSI. -p=4 có nghĩa là chạy tối đa bốn build song song.
