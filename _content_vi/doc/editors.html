Giới thiệu

Tài liệu này liệt kê các plugin trình soạn thảo và IDE thường dùng từ hệ sinh thái Go giúp việc lập trình Go hiệu quả và liền mạch hơn. Danh sách đầy đủ về hỗ trợ trình soạn thảo và IDE cho việc lập trình Go có tại trang wiki.

Các lựa chọn

Hệ sinh thái Go cung cấp nhiều plugin trình soạn thảo và IDE để nâng cao trải nghiệm chỉnh sửa, điều hướng, kiểm thử và gỡ lỗi hàng ngày của bạn.

  • Visual Studio Code: Plugin Go cung cấp hỗ trợ ngôn ngữ lập trình Go
  • GoLand: GoLand được phân phối dưới dạng IDE độc lập hoặc dưới dạng plugin cho IntelliJ IDEA Ultimate
  • vim: plugin vim-go cung cấp hỗ trợ ngôn ngữ lập trình Go

Lưu ý rằng đây chỉ là một vài giải pháp hàng đầu; danh sách đầy đủ hơn được cộng đồng duy trì về IDE và plugin trình soạn thảo có tại Wiki.
