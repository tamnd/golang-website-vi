Tải xuống và cài đặt Go nhanh chóng với các bước được mô tả ở đây.

Để tìm hiểu thêm về cài đặt, bạn có thể quan tâm đến:

  • Quản lý cài đặt Go -- Cách cài đặt nhiều phiên bản và gỡ cài đặt.
  • Cài đặt Go từ mã nguồn -- Cách kiểm tra mã nguồn, xây dựng chúng trên máy của bạn và chạy chúng.

Tải xuống

Cài đặt Go

Chọn tab hệ điều hành của máy tính bên dưới, sau đó làm theo hướng dẫn cài đặt của nó.

  1. Xóa bất kỳ cài đặt Go trước đó bằng cách xóa thư mục /usr/local/go (nếu có), sau đó giải nén file lưu trữ bạn vừa tải xuống vào /usr/local, tạo cây Go mới trong /usr/local/go: 
          $ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.14.3.linux-amd64.tar.gz

    (Bạn có thể cần chạy từng lệnh riêng lẻ với các quyền cần thiết, với tư cách root hoặc qua sudo.)

    Không giải nén file lưu trữ vào một cây /usr/local/go hiện có. Điều này được biết đến là gây ra các cài đặt Go bị hỏng.

  2. Thêm /usr/local/go/bin vào biến môi trường PATH.

    Bạn có thể làm điều này bằng cách thêm dòng sau vào $HOME/.profile hoặc /etc/profile (để cài đặt toàn hệ thống): 

              export PATH=$PATH:/usr/local/go/bin

    Lưu ý: Các thay đổi đối với tệp profile có thể không được áp dụng cho đến lần đăng nhập tiếp theo vào máy tính của bạn. Để áp dụng các thay đổi ngay lập tức, hãy chạy các lệnh shell trực tiếp hoặc thực thi chúng từ profile bằng một lệnh như source $HOME/.profile.

  3. Xác minh rằng bạn đã cài đặt Go bằng cách mở command prompt và gõ lệnh sau: 
              $ go version
  4. Xác nhận rằng lệnh in ra phiên bản Go đã cài đặt.
  1. Mở tệp package bạn đã tải xuống và làm theo các hướng dẫn để cài đặt Go.

    Package cài đặt bản phân phối Go vào /usr/local/go. Package sẽ đặt thư mục /usr/local/go/bin vào biến môi trường PATH của bạn. Bạn có thể cần khởi động lại bất kỳ phiên Terminal đang mở nào để thay đổi có hiệu lực.

  2. Xác minh rằng bạn đã cài đặt Go bằng cách mở command prompt và gõ lệnh sau: 
              $ go version
  3. Xác nhận rằng lệnh in ra phiên bản Go đã cài đặt.
  1. Mở tệp MSI bạn đã tải xuống và làm theo các hướng dẫn để cài đặt Go.

    Theo mặc định, trình cài đặt sẽ cài đặt Go vào Program Files hoặc Program Files (x86). Bạn có thể thay đổi vị trí nếu cần. Sau khi cài đặt, bạn cần đóng và mở lại bất kỳ command prompt đang mở nào để các thay đổi môi trường do trình cài đặt thực hiện được phản ánh trong command prompt.

  2. Xác minh rằng bạn đã cài đặt Go.
    1. Trong Windows, nhấp vào menu Start.
    2. Trong hộp tìm kiếm của menu, gõ cmd, sau đó nhấn phím Enter.
    3. Trong cửa sổ Command Prompt xuất hiện, gõ lệnh sau: 
                    $ go version
    4. Xác nhận rằng lệnh in ra phiên bản Go đã cài đặt.

Đã sẵn sàng!
